User guide for AppQuiz

A guide to help the user understand the AppQuiz .

  1. 1. Location of user is tracked (given GPS settings are turned on users phone), a radius of 50m is also shown around the tracked location as well the question points being loaded on to the map.
  2. 2. If location user is not automatically tracked - Click track location to location users position on the leaflet map
  3. 3. Alongside automatically loading question points - users can manually load them by pressing 'Get question points' in navigation menu.
  4. 4. The user location point can be identified by clicking the central point within the blue radius circle. When clicked on, a pop-up will appear, displaying "You Are Here".
  5. 5. Each question point is identified by a location name e.g. UCL entrance
  6. 6. Users can also calculate the distance between their current located position to warren street station, in kilometers.
  7. 7. Once a question point is within the users 50m radius - users can click 'Answer question' in the navigation menu. This would be then followed by clicking the relevant question point to answer the question.
  8. 8. Once the question point is clicked - the page will change to show the quiz question - where the user has to select an answer from the given options - once this is submitted an alert will appear telling the user if the answer is correct or incorrect.
  9. 9. If the question point is not within the users radius - and the user tries to click the question - the app will calculate the users distance and if it is outside of the given distance an alert will appear telling the user to get closer to the question marker.
  10. Repeat the above process for each question - until all questions are answered!.
